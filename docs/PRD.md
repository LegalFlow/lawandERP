# 법무법인 로앤 법률컨설팅팀 실적 관리 시스템 PRD
## 1. 프로젝트 개요
법무법인 로앤의 법률컨설팅팀 실적 관리 시스템은 구성원들의 업무 실적을 효율적으로 관리하고 분석하기 위한 내부 도구입니다. 이 시스템은 구성원 관리, 영업팀의 상담 및 계약 현황, 은행 및 CMS 거래 내역, 직접 입력 매출, 그리고 대시보드를 통해 실적을 시각화합니다. 이를 통해 관리자는 팀의 성과를 한눈에 파악하고, 구성원들은 자신의 업무 현황을 쉽게 확인할 수 있습니다.

## 2. 유저 플로우
### 사용자 역할
#### 관리자
- 모든 메뉴와 기능에 접근 가능
- 구성원 관리 및 데이터 수정 권한 보유
#### 일반 사용자
- 권한에 따라 제한된 메뉴에만 접근 가능
- 자신의 실적 및 관련 데이터 조회 가능
### 플로우 상세
#### 로그인
- 사용자는 아이디와 비밀번호로 로그인합니다.
- 권한에 따라 접근 가능한 메뉴가 설정됩니다.
#### 대시보드
- 로그인 후 가장 먼저 대시보드가 표시됩니다.
- 법률컨설팅팀의 당월 실적이 리스트로 표시됩니다.
#### 메뉴 네비게이션
- 구성원 관리 (관리자만)
  - 구성원 등록, 수정, 삭제
- 영업팀 실적 관리 시스템
  - 신규상담
    - 오늘 날짜 기준 상담 리스트 확인
    - 필터를 통해 기간, 고객명, 지역, 담당자별 조회
  - 계약현황
    - 진행된 계약 리스트 확인
    - 필터를 통해 기간, 소속, 담당자, 고객명별 조회
  - 은행 거래내역
    - 자동으로 불러온 은행 입금 내역 확인
    - 필터를 통해 기간, 입금자명, 계정, 담당자별 조회
  - CMS 거래내역
    - 자동으로 불러온 CMS 입금 내역 확인
    -   은행 거래내역과 동일한 방식으로 작동
  - 매출 직접입력
    - 신규 매출 내역 직접 입력
    - 필터를 통해 다양한 조건으로 조회
#### 데이터 입력 및 수정
- 각 메뉴에서 필요한 데이터를 입력하거나 수정합니다.
- 자동저장이 기본으로 지원되며, 좌측 상단에 저장 상태가 표시됩니다.
#### 보고서 생성
- 주별, 월별 등 지정된 기간이 끝난 후 GPT를 활용하여 실적 보고서를 생성합니다.
- 보고서를 기반으로 인사고과 등급이 부여됩니다.
## 3. 핵심 기능
### 구성원 관리
#### 구성원 등록 및 삭제
- 신규 구성원 정보를 입력하여 등록
- 기존 구성원 삭제 기능
#### 필드
- 이름, 직급, 업무, 소속, 상태, 권한, 년차, 주당 근무일수, 주당 재택근무일수, 은행, 계좌번호, 기타사항
#### 특이사항
- 드롭다운 및 텍스트 필드 활용
- 모든 입력 필드는 탭 키로 이동 가능
### 영업팀 실적 관리 시스템
#### 신규상담
- 기능
  - 특정 일자의 상담 리스트 표시
  - 메시지 큐를 통해 외부에서 데이터 수신
  - 진행현황에 따라 계약일 자동 입력
#### 필터링 도구
  - 기간 선택, 고객명 검색, 지역 선택, 담당자 선택, 오늘 버튼
#### 대시보드 정보
- 총 상담수, 지역별 상담수, 계약수, 계약률
#### 계약현황
- 기능
  - 계약 진행된 리스트 표시
  - 계약일 기준 월별 리스트 표시
- 필터링 도구
  - 기간 선택, 소속 선택, 담당자 이름 검색, 고객명 검색
- 요약 정보
  - 총 수임료, 평균 수임료
#### 은행 거래내역
- 기능
  - 효성 가상계좌의 결제내역 자동 수신
  - 입금 내역 확인 및 담당자, 계정, 메모 수정
- 필터링 도구
  - 기간 선택, 입금자명 검색, 계정 검색, 담당자 검색
#### CMS 거래내역
-  기능
  - CMS 결제내역 자동 수신
  - 은행 거래내역과 동일한 방식
#### 매출 직접입력
- 기능
  - 신규 매출 내역 직접 등록
  - 모든 필드 수정 가능
- 필터링 도구
  - 은행 거래내역과 동일
### 대시보드
#### 기능
- 법률컨설팅팀 구성원들의 당월 실적 리스트 표시
#### 필드
- 구성원 이름, 현금매출액, 계약서매출액, 계약률
#### 정렬 기능
- 필드명을 클릭하여 내림차순/오름차순 정렬
## 4. 기술 스택
초보 개발자를 고려하여 다음과 같은 기술 스택을 추천합니다.
#### 프론트엔드
- HTML/CSS: 기본적인 마크업과 스타일링
- JavaScript: 기본적인 인터랙션 구현
- React.js 또는 Vue.js: 사용자 인터페이스 구축에 용이한 프레임워크
- React는 비교적 커뮤니티와 자료가 풍부하며, Vue.js는 러닝 커브가 낮음
#### 백엔드
- Node.js와 Express.js: JavaScript 기반 서버 사이드 개발에 적합
- Python과 Django 또는 Flask: 간단하고 빠른 웹 개발 가능
- Python은 문법이 쉬워 초보자에게 적합
#### 데이터베이스
- SQLite 또는 PostgreSQL: 설정이 간단하고 사용이 쉬움
#### 기타
- Git: 버전 관리
- Bootstrap 또는 Tailwind CSS: UI 라이브러리 활용으로 빠른 개발
API 통신: 은행 및 CMS 거래내역 수신을 위한 RESTful API 사용
자동저장 구현을 위한 라이브러리: 예를 들어, Axios를 사용하여 비동기 요청 처리
## 5. MVP 기능 개발 이후 추가 개선사항
#### 실적 보고서 자동화
- GPT를 활용한 실적 분석 및 보고서 자동 생성 기능 구현
#### 인사고과 시스템
- 실적 보고서를 기반으로 한 자동 인사고과 등급 부여 시스템 개발
#### 권한 관리 세분화
- 관리자와 일반 사용자 외에 역할별 권한 설정 기능 추가
#### UI/UX 개선
- 사용자 경험을 향상시키기 위한 인터페이스 개선
#### 모바일 대응
- 반응형 웹 디자인 적용으로 모바일 환경 지원
#### 알림 시스템
- 주요 이벤트 발생 시 이메일 또는 앱 내 알림 기능 추가
#### 데이터 시각화
- 차트나 그래프를 활용한 실적 데이터 시각화
#### 주의사항
- 자동저장 기능 구현 시 저장 상태를 모든 페이지의 좌측 상단에 표시해야 합니다.
- 키보드 네비게이션을 지원하여 모든 입력 필드 간 탭 이동이 가능하도록 합니다.
- 라이브러리 활용 극대화를 통해 개발 효율성을 높입니다.
